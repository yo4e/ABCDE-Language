# ABCDE Adaptations

ABCDE can be used as-is, but by **replacing the writing system**, it can fit more naturally into the worldbuilding of fantasy fiction, games, and TRPGs.

This page shows adaptation ideas that change only the **visual appearance** of ABCDE while keeping its **meaning and grammar** intact.

## 1. Symbol Substitution (Fantasy Script)

### Base Mapping

- `A` -> `△`
- `B` -> `○`
- `C` -> `□`
- `D` -> `×`
- `E` -> `▽`

### How the Substitution Works

- Convert an ABCDE string symbol-by-symbol, one character at a time.
- Grammar, meaning, word order, and omission rules remain **the same as standard ABCDE**.
- In other words, this is not a "new language," but an **alternate writing style (decorative orthography)** for ABCDE.

## 5. Ideas for Use in Creative Works

- **Ruins script**: Use it on stone tablets, doors, or map fragments
- **Magic language**: Use it for chants, seals, or incantations
- **Non-trade language**: A shared language of ancients, pilgrims, or forest people
- **UI flavor**: Use it in signs, quest text, or puzzle gimmicks in games

## 6. Practical Tips (Readability)

- On first use, show both `ABCDE spelling` and `symbol spelling`
- Avoid writing long passages entirely in symbols; use short phrases and set expressions
- Provide a one-page mapping chart for players/readers

## 7. Notes (Relation to the Canonical Orthography)

- The **canonical orthography (specification form)** is **ABCDE**
- Symbol notation is intended as a visual/worldbuilding adaptation
- External words like `[Tokyo]` can remain as-is, or be converted using your own in-world rules

Example:

- `E? [Tokyo] ACE`
- `▽? [Tokyo] △□▽`

## 8. Try Making a Derived Language

ABCDE is well-suited to a model where you **share a minimal core specification** and freely expand it in derived forms.

Feel free to create a derived language for your own work, setting, or use case.

For example:

- Change only the script (symbols, runes, decorative letters)
- Add more set phrases (religious, military, merchant, traveler, etc.)
- Add setting-specific vocabulary (place names, magic, creatures, culture)
- Introduce dialect differences in pronunciation rules

Tips:

- If you preserve the original ABCDE meaning/grammar, mutual learning stays easy
- Large changes are also fine (just document the "derived rules" clearly)
- A mapping chart and short example phrases help readers/players learn quickly
