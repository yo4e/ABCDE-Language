# ABCDE Specification (survival focus)

Name pronunciation (Japanese reading): アベケデ / IPA `[a.be.ke.de]`

ABCDE is an ultra-minimalist language designed for **basic human connection and survival**, not for commerce or complex abstract systems. All meaning is derived from **8 Prime Atoms**.

## 1. The 8 Atomic Syllables
ABCDE is built from 8 core sounds. Pronunciation is designed to be easy for speakers of any language.

| Atom | IPA | Sounds like | Meaning |
| :--- | :--- | :--- | :--- |
| **A** | [a] | **A**h (father) | Future / Motion |
| **E** | [e] | **E**h (pet) | Past / Relation |
| **BA** | [ba] | **Ba**r | Self / Origin |
| **BE** | [be] | **Be**nd | Other / Target |
| **CA** | [ka] | **Ka**yak | Matter / Fact |
| **CE** | [ke] | **Ke**ttle | Positive / Good |
| **DA** | [da] | **Da**rk | Action / Power |
| **DE** | [de] | **De**sk | Negative / Bad |

- **C** is always pronounced as **[k]** (as in "cat"), never as [s].
- **Vowels** are always short and clear.

### 1.1 Character Set (Orthography)
In normal ABCDE writing (outside square brackets), use only:
- uppercase Latin letters `A B C D E`
- Arabic numerals `0-9`
- space and hyphen `-`
- `!` and `?`
- square brackets `[ ]` (for external words / proper nouns)

Inside `[ ]`, external words may be written in their original spelling/script.

## 2. Compounding Logic (Agglutination)
**Rule**: Modifier + Head (The main category is always the LAST atom).
The last atom dictates the "kind" of thing it is.

- `CECA`: Good-thing = Food / Treasure
- `BACA`: Self-thing = Human / Person
- `ACA`: Day / Red / Place
- `BADA`: Step (Self-action)
- `EDA`: Logic-action = Think / Calculate

### 2.1 Negation Prefix (`DE-`)
When `DE` is used as a **grammatical "not"** (to negate a verb or statement), it must be written as the prefix `DE-` and attached directly to the next word (**no space**).

Plain `DE` may still be used as a normal atom inside compounds or as a lexical negative word (`DE`, `DEDE`, `DECA`, etc.).

- `DECE`: Bad-good = Sad
- `DE-CE`: Not good
- `DE-DA`: Don't do / Not do
- `DE-BACA`: Not a person

### 2.2 Reduplication (Repetition)
Repeating a word or atom increases its quantity or intensity.

- **Plurality**: `BACA BACA` (People / Humans), `CECA CECA` (Much food / Feast)
- **Intensity**: `CECE` (Very good), `DEDE` (Very bad / Terrible), `DADA DADA` (Huge / Giant)

### 2.3 Context Omission (Casual Speech)
In common greetings or when the context is undeniable, not only the subject (`BA` or `BE`) but also other recoverable elements (object, place, time) may be omitted.

- `BA CE BE` (I like you) -> `CE BE` (Thank you / Good you)
- `BA ADA ACA` (I come here) -> `ADA ACA` (Coming here)
- `E? BE CECA CE` (Is your food good?) -> `E? CECA CE` (Food good?)

### 2.4 Recommended Word Order (Flexible)
Word order is basically flexible. For learning and listening clarity, the recommended default pattern is:

`(Sentence markers) -> (Subject/Topic) -> (Predicate) -> (Object) -> (Place) -> (Time)`

You may front an element for emphasis. If meaning becomes unclear, return to the default pattern.

## 3. Number System & Measurements
ABCDE uses Arabic numerals without commas (`1000000`). It rejects abstract metric/imperial units in favor of **Natural References**.

### Numerical Pronunciation (Arabic Numerals)
Numbers are pronounced using specific phonetic sounds to ensure they are distinct from the 8 prime atoms.

### Non-Commercial Logic
There are no words for "money", "price", "buy", or "tax". Communication focuses on sharing, needing, and making.

### Natural Units
- **Time (1 Day)**: `ACA` (Sun-thing / Red)
- **Distance (1 Step)**: `BADA` (Self-action)
- **Small Size (1 Egg)**: `DEDECECA` (Small-food; also usable as "egg" by context)

- `3ACA`: 3 Days
- `1000BADA`: 1000 Steps

## 4. Tense & Aspect
Tense is indicated by particles placed at the **beginning of the sentence** (before the clause/predicate).

Scope rule: `A-` and `E-` are **sentence-level markers** (sentence-initial), while `DE-` is a **word-level negation prefix** attached to the specific word it negates.
Recommended order rule: if a mood marker (`E?` / `A!`) and a tense marker (`A-` / `E-`) are both present, place the mood marker first for clarity (e.g., `E? A- BA ADA ACA`).

| Particle | Tense | Meaning |
| :--- | :--- | :--- |
| (None) | Present | Current / General state |
| **E-** | Past | Relation / Completed action |
| **A-** | Future | Motion / Intended action |

- `E- BA DA CA`: I did it (Past-I-do-thing)
- `A- BA DA CA`: I will do it (Future-I-do-thing)

## 5. Pronunciation Guidelines
To reduce listening errors:
- **Stress**: Always stress the *last* syllable of a word (e.g., Ba-BÉ, Ce-CÁ).
- **Separation**: Leave a clear space between words. Commas and periods are **not used**.
- **Emotion/Logic**: The default form is `E?` at the **beginning** for questions and `A!` at the **beginning** for emphasis/exclamations. In fast speech, short exclamatory forms like `CE!` / `DE!` are also acceptable.
- **Marker Phonetics (Short vs Long)**:
  - **Exclamation/Question (`A!`, `E?`)**: Pronounce with a **short, sharp** sound.
  - **Grammatical Markers (`A-`, `E-`, `DE-`)**: Pronounce with a **long, extended** vowel (e.g., Aaa, Eee, Deee) to distinguish from standard atoms.
- **Vowels**: Standard atoms are always short and clear.

## 6. External Concept Rule (Irregular)
When a concept does not exist in ABCDE (e.g., proper nouns, names, complex technology), you may insert an external word in square brackets `[ ]`.

### Rules
- **Grammar**: The bracketed word acts as a `CA` (Thing) or `BA-CA` (Person).
- **Preference**: Use ABCDE atoms whenever possible. Only use brackets as a last resort.
- **Proper Nouns**: Always bracketed.

- E? [Tokyo] ACE -> Tokyo is a good place / paradise?
- A! BA [Smartphone] CE -> I really like my smartphone!
- BACA [Tanaka] CE -> Person Tanaka is good
