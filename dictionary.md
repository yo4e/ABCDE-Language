### Numbers
- **0 - 9**: Use standard Arabic numerals. 
- **Spoken**: 0:ZERO, 1:WAN, 2:TSU, 3:SURI, 4:HOO, 5:HAI, 6:SIKU, 7:SEFU, 8:ETO, 9:NAI.
- Place digits before the noun.

### Natural Units (Measurements)
- **ACA**: Day (Red/Place/Sun-thing)
- **BADA**: Step (Self-action)
- **DEDECECA**: Egg / Egg-size (object or small size unit, by context)
- (No words for money, price, or abstract units like meters/grams)

### Core People
- **BA**: I
- **BE**: You
- **BABA**: We (Self + Self)
- **BABE**: Friend (Self + Other)

### Core States
- **CE**: Good / Yes / Like
- **DE**: Bad / No (lexical negative atom; also used in compounds like `DEDE`, `DECA`)
- **DE-**: Grammatical negation prefix (Not) — attach directly to the next word (e.g., `DE-CE`, `DE-DA`)
- **E-**: Past tense marker
- **A-**: Future tense marker
- **Word Word**: Plural or intensified (e.g., `BACA BACA` = people)
- **CECE**: Very good / Perfect
- **DEDE**: Very bad / Terrible

### Core Objects
- **CA**: Thing / It
- **CECA**: Food (Good-thing)
- **DEDECECA**: Egg (Small-food); also used as an egg-size unit
- **DECA**: Poison / Dirt (Bad-thing)
- **DACA**: Tool (Action-thing)
- **BACA**: Person (Self-thing/Human)

### Core Actions
- **DA**: Do / Make
- **ADA**: Go / Walk (Movement-do)
- **EDA**: Think (Logic-do)
- **CEDA**: Help (Good-do)
- **DEDA**: Break / Hurt (Bad-do)

### Core Places
- **ACA**: Here (At-thing)
- **ABE**: There (At-other)
- **ACE**: Paradise / Good place (At-good)
- **ADE**: Hell (At-bad)

### Nature & Complex Concepts (Decomposition)
- **BADACA**: Plant (Life-thing)
- **DADA BADACA**: Tree (Big plant)
- **DEDE BADACA**: **Leaf** (Small plant)
- **CE BADACA**: Flower (Good plant)
- **DADA CA**: Mountain (Big thing)
- **DEDE CA**: Stone / Grain (Small thing)
- **CEA CA**: Sun (Good-motion thing / Light source)
- **ACA**: Day / Red (Sun/Fire-thing / Target-place)
- **BADA**: Step (Self-action)
- **ADA**: Water / Drink / Blue (Motion-action)
- **ECA**: Yellow (Light/Grain-thing)

### Size & Intensity
- **DADA**: Big / Great / Power
- **DEDE**: Small / Little / Weak
- **DADA DADA**: Huge / Giant
- **DEDE DEDE**: Tiny / Micro

### Colors (Natural references)
- **ACA**: Red (Active-thing / Sun-place)
- **ADA**: Blue (Flowing-action / Water)
- **ECA**: Yellow (Light-thing / Grain)
- **A**: White / Light / Bright
- **DEA**: Black / Dark / Night
